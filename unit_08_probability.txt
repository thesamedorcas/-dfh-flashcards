probability; a number in [0, 1] measuring how likely an event is — 0 = impossible, 0.5 = equally likely either way, 1 = certain
sample space; the set of all possible outcomes — e.g. for a die roll: S = {1, 2, 3, 4, 5, 6}
frequency; how often an event occurs in repeated trials — e.g. heads appeared 52 times out of 100 flips → frequency = 0.52
outcomes; the individual possible results — e.g. rolling a die: outcomes are 1, 2, 3, 4, 5, 6 (each is a single outcome)
samples; a collection of observed outcomes — e.g. [3, 1, 4, 1, 5, 9, 2, 6] is a sample of 8 dice rolls
events; subsets of the sample space — e.g. "roll an even number" = {2, 4, 6}; every outcome is an event, but not every event is a single outcome
example of event; rolling a die and getting an even number → event = {2, 4, 6}; P(even) = 3/6 = 0.5
example of an outcome; rolling a die and getting a 2 — a single specific result
distributions; functions that assign probabilities to outcomes — e.g. a fair die: P(X=k) = 1/6 for k ∈ {1,...,6}
P(X=x) notation; probability that random variable X takes the specific value x → e.g. P(die=3) = 1/6 ≈ 0.167
P(A) notation; probability of event A occurring → e.g. P({roll > 4}) = P({5,6}) = 2/6 ≈ 0.333
P(A|B) notation; probability of A given that B has already occurred → e.g. P(even | roll > 3) = P({4,6}) / P({4,5,6}) = 2/3
P(A ∩ B) notation; probability of both A AND B occurring → e.g. P(even AND > 3) = P({4,6}) = 2/6 ≈ 0.333
P(A ∪ B) notation; probability of A OR B (or both) → P(A∪B) = P(A) + P(B) - P(A∩B)
random variable; a variable whose value is the outcome of a random process → e.g. X = number on a die roll
discrete random variable; takes countable distinct values → e.g. number of heads in 10 flips: X ∈ {0,1,...,10}
continuous random variable; can take any value in a range → e.g. height of a random person: X ∈ [0, ∞)
probability mass function (PMF); gives exact probability for each discrete value → e.g. fair coin: P(X=0) = 0.5, P(X=1) = 0.5; all values sum to 1
probability density function (PDF); gives probability density for continuous values — area under curve in a range = probability → e.g. PDF of N(0,1) at x=0 is ≈ 0.399 (density, not probability)
density; a non-negative value measuring probability concentration — NOT a probability itself; can exceed 1 → P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx
axioms of probability; three fundamental rules: (1) P(A) ≥ 0; (2) P(S) = 1; (3) P(A∪B) = P(A)+P(B) for mutually exclusive A,B
axiom of unitarity; the probability of the entire sample space is 1 → P(S) = 1 — something must happen
axiom of non-negativity; all probabilities are ≥ 0 — you can't have negative probability
axiom of additivity (sum rule); for mutually exclusive events: P(A∪B) = P(A) + P(B) → e.g. P(1 or 2) = 1/6 + 1/6 = 2/6
conditional probability; P(A|B) = probability of A given B happened → narrows the sample space to B → P(A|B) = P(A∩B) / P(B)
conditional probability from a PMF; P(X=x | Y=y) = P(X=x, Y=y) / P(Y=y) — the joint probability normalised by the evidence
forward probability; predict outcomes from known parameters → e.g. "given a fair coin, what's P(3 heads in 5 flips)?"
inverse probability; infer parameters from observed data → e.g. "given I saw 3 heads in 5 flips, what is P(coin is fair)?"
Bayesian interpretation of probability; probability = degree of belief — can be updated with new evidence via Bayes' rule; allows non-repeatable events to have probabilities
frequentist interpretation of probability; probability = long-run relative frequency — based on repeating an experiment many times
Bayes' Rule equation; $P(A|B) = \dfrac{P(B|A)\,P(A)}{P(B)}$ — posterior ∝ likelihood × prior
Bayes' Rule interpretation; update your prior belief P(A) using the likelihood of the evidence P(B|A) to get the posterior P(A|B)
Posterior; P(A|B) — your updated belief about A after observing evidence B
Likelihood; P(B|A) — how probable the evidence B is if A is true — used to evaluate how well a model explains the data
Prior; P(A) — your belief about A before seeing any evidence
Evidence; P(B) — the total probability of observing the evidence under all hypotheses: $P(B) = \sum_a P(B|A=a)P(A=a)$
Marginal Likelihood; P(B) — normalising constant in Bayes' rule — ensures the posterior is a valid probability distribution
Integration over the evidence in Bayes' Rule; $P(B) = \sum_a P(B|A=a)P(A=a)$ — sum/integrate likelihood×prior over all possible values of A
joint distribution; P(X,Y) — probability of each combination of values for X and Y → e.g. P(die1=3, die2=4) = (1/6)(1/6) = 1/36 if independent
marginal distribution; P(X) obtained by summing/integrating out Y from the joint: $P(X=x) = \sum_y P(X=x, Y=y)$
conditional distribution; P(X|Y=y) — the distribution of X when Y is fixed at y → obtained by slicing the joint at Y=y and renormalising
marginalization formula; to get P(X), sum the joint over all values of Y: $P(X=x) = \sum_y P(X=x,\, Y=y)$
conditioning formula; $P(X|Y) = P(X,Y) / P(Y)$ — divide the joint by the marginal to condition on Y
bigrams; pairs of consecutive elements — e.g. in "hello": bigrams are (h,e),(e,l),(l,l),(l,o); used in language models
n-gram models; predict the next token from the previous n−1 tokens using conditional probabilities → P(word | previous n−1 words)
likelihood; P(data | parameters) — how probable the observed data is given specific parameter values
likelihood function; treat the data as fixed and vary the parameters → L(θ|data) = P(data|θ); maximise this to find the best θ
likelihood of multiple independent events; $L = \prod_i P(x_i|\theta)$ — multiply individual probabilities (or sum log-probabilities)
log-likelihood; $\log L(\theta) = \sum_i \log P(x_i|\theta)$ — turns products into sums, avoids floating-point underflow for many small probabilities
expectation; the long-run average value of a random variable over many repetitions
intuitive meaning of expectation; if you repeat the experiment infinitely many times, E[X] is the average of all outcomes — e.g. E[fair die] = (1+2+3+4+5+6)/6 = 3.5
expected value; $E[X] = \sum_x x\,P(X=x)$ (discrete) or $E[X] = \int x\,f(x)\,dx$ (continuous) — the probability-weighted average value
expectation formula (for a discrete random variable); $E[X] = \sum_x x\,P(X=x)$ — e.g. fair die: $E[X] = 1{\cdot}\tfrac{1}{6} + 2{\cdot}\tfrac{1}{6}+\ldots+6{\cdot}\tfrac{1}{6} = 3.5$
expectation formula (for a continuous random variable); $E[X] = \int_{-\infty}^{\infty} x\,f(x)\,dx$ — weighted by the PDF
expectation of a function of a random variable; $E[g(X)] = \sum_x g(x)\,P(X=x)$ — e.g. $E[X^2]$ for a fair die = $(1^2+2^2+\ldots+6^2)/6 = 91/6 \approx 15.2$
entropy; $H(X) = -\sum_x P(X=x)\log_2 P(X=x)$ — measures uncertainty in bits → a fair coin has H=1 bit; a biased coin with P(H)=0.99 has H≈0.08 bits
entropy equation (for discrete random variables); $H(X) = -\sum_x P(x)\log_2 P(x)$ — uniform distribution has the highest entropy (most uncertain)
entropy interpretation; high entropy = lots of uncertainty (hard to predict); low entropy = little uncertainty → e.g. a loaded die (always lands 6) has entropy 0
information (in bits); $-\log_2 P(x)$ — how surprising outcome x is → rare events carry more information: P=1/128 → 7 bits of information
bit; one binary choice (0 or 1) — a fair coin flip provides exactly 1 bit of information
entropy as average information; H(X) = expected value of $-\log_2 P(X)$ — average surprise over all outcomes
entropy as expected log probability; $H(X) = -E[\log_2 P(X)]$ — the average number of bits needed to describe a random outcome
information theory; the mathematical study of encoding, storing, and transmitting information — founded by Shannon; entropy is the central measure
logits; $\text{logit}(p) = \log(p/(1-p))$ — log-odds; maps (0,1) probabilities to (−∞, +∞) → logit(0.5)=0, logit(0.9)≈2.2, logit(0.1)≈−2.2
odds; P(event)/P(not event) → e.g. P=0.75 → odds = 0.75/0.25 = 3 ("3 to 1")
log-probabilities; $\log P(x)$ — always ≤ 0; turn products into sums; prevent underflow when multiplying many small probabilities together
log-odds; $\log(P/(1-P))$ — the logit; used in logistic regression; 0 = even odds, positive = more likely, negative = less likely
logistic function; $\sigma(z) = 1/(1+e^{-z})$ — maps any real number to (0,1) → $\sigma(0)=0.5$, $\sigma(\infty)=1$, $\sigma(-\infty)=0$
empirical distribution; $P(X=x) = \text{count}(x)/N$ — the histogram of observed data normalised to sum to 1
sampling from distributions; drawing random values X ~ F(x) — e.g. np.random.normal(0,1) draws samples from N(0,1)
Monte Carlo approximation to expectation; $E[g(X)] \approx \frac{1}{N}\sum_{i=1}^N g(x_i)$ where $x_i \sim P(X)$ — average g over many random samples
empirical distribution; \( P(X=x) = \frac{\text{count}(x)}{N} \) — the normalised frequency histogram of observations
sampling from distributions; \( X \sim F(x) \) — draw random samples from distribution F; e.g. rng.normal(0,1, size=1000)
Monte Carlo approximation to expectation; \( E[g(X)] \approx \frac{1}{N} \sum_{i=1}^{N} g(x_i) \) — e.g. estimate P(X² > 2) for X~N(0,1) by drawing 10000 samples and computing the fraction satisfying x²>2
Monte Carlo error; error ∝ $1/\sqrt{N}$ — 100× more samples → 10× more accurate; fundamentally slow convergence
convergence of Monte Carlo methods; as N→∞, the sample average converges to the true expectation — guaranteed by the Law of Large Numbers
What is the frequentist interpretation of probability?;probability = long-run relative frequency of an event over many identical trials → P(heads) = lim_{N→∞} (# heads) / N
What is the Bayesian interpretation of probability?;probability = degree of belief — can be updated using Bayes' rule when new evidence arrives; doesn't require repeatable experiments
What is a generative model?;a model of the process that produces data — can run forward (simulate data) or backward (infer parameters from data)
What is forward probability?;predict outcomes from known parameters — e.g. "if the coin is biased at 0.6, what is P(7 heads in 10 flips)?"
What is inverse probability?;infer parameters from observed outcomes — e.g. "I saw 7 heads in 10 flips; what is P(bias=0.6)?"
What is the sum rule of probability?;P(A or B) = P(A) + P(B) − P(A and B) — the overlap is subtracted to avoid double-counting
What is the definition of conditional probability?;P(A|B) = P(A and B) / P(B) — restricts the sample space to only cases where B occurred
What is marginalisation?;summing a joint distribution over one variable to get the marginal: $P(X) = \sum_y P(X,Y=y)$ — "integrating out" Y
What are independent random variables?;X ⊥ Y if P(X,Y) = P(X)P(Y) — knowing Y gives you no information about X → e.g. two separate fair dice
What is a bigram model?;P(c_i | c_{i−1}) — probability of each character given the previous one; e.g. after 'q', 'u' is very likely in English
What is the natural frequency method for Bayes?;imagine 100,000 people, count subgroups → avoids abstract probabilities; e.g. 1 has disease, 0.95 test positive, 4,999 false positives → only 1/5000 positives are true
What causes the apparent paradox in the Plague X example?;even a 95%-accurate test gives mostly false positives when prevalence is 1/100,000 — the tiny prior crushes the posterior
What are log-probabilities used for computationally?;prevent underflow: instead of $\prod_i P(x_i) \approx 0$, compute $\sum_i \log P(x_i)$ — sums of logs are numerically stable
What is the empirical distribution and how is it computed?;$\hat{P}(X=x) = \text{count}(x)/N$ — the histogram normalised by total count; only meaningful for discrete variables
Why does the empirical distribution not work for continuous random variables?;each continuous value is observed at most once — count=1 for every point → normalised histogram is meaningless; need density estimation (e.g. KDE)
