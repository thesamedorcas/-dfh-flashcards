# DF(H) Flashcards

A browser-based flashcard app for the DF(H) course, covering units from numerical methods to signals and probability.

## Usage

Open `index.html` in any modern browser. Cards are fetched from the [df_h_flashcards](https://github.com/johnhw/df_h_flashcards) GitHub repository — an internet connection is required.

## Features

- **Unit filtering** — switch between all cards or a specific unit via the tab strip
- **Flip animation** — click the card, press the Flip button, or hit `Space` / `Enter`
- **Keyboard navigation** — `←` / `→` arrow keys to move between cards
- **Swipe support** — swipe left/right on touch devices
- **Progress bar** — shows how far through the current deck you are

## Card format

Cards are plain `.txt` files with one card per line, formatted as:

```
question ; answer
```
